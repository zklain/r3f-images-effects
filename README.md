## R3F Effects

This repo contains examples of various 3D effects implemented with [react-three-fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/examples)

### Deps

- vite
- React@18 w. TS
- @react-three/fiber
- @react-three/drei
- three
- leva
- @react-spring/three

### Commands

`npm install && npm run dev` is all you need for now
